# 🧠 全息量化系統：AI 籌碼深度診斷說明書 (V60 核心大腦)

「AI 全息籌碼深度診斷總結」是本系統的最高決策核心。它將龐雜的技術線型、集保戶數、分點進出、以及董監死籌碼，透過多維度矩陣運算後，濃縮成五個層次的戰情匯報。

以下是各階層的深度解析與實戰應用指南：

## 👁️ 第零層：幾何形態與結構 (AI 視覺辨識)
**【核心任務】**：判斷目前的 K 線圖形處於什麼樣的物理結構中，給出突破或跌破的預警。
* **觸發形態**：系統會自動掃描近期的 K 線，辨識出標準的技術形態（例如：W底、M頭、頭肩頂/底、上升/下降三角形、深 V 反轉等）。
* **實戰解讀**：
    * **偏多結構（Bullish）**：如 W 底、頭肩底。這代表股價有強烈的築底企圖。若此時配合「聰明錢流入」，通常是絕佳的左側建倉點或右側突破點。
    * **偏空結構（Bearish）**：如 M 頭、頭肩頂。這代表上方累積了沉重的套牢賣壓。若出現此形態且主力正在撤退，絕對不能有抄底的幻想。
    * **收斂整理**：如對稱三角形、箱型矩形。代表多空勢均力敵，即將表態。此時需緊盯「第二層」與「第三層」的籌碼流向，籌碼往哪走，突破方向就在哪。

## 🛡️ 第一層：長線底盤與動態通道 (防守線與價格重心)
**【核心任務】**：看透長線主力的底牌，確認目前的股價是處於「安全區」還是「危險區」。
* **主力成本乖離率**：系統會濾除當沖與極短線雜訊，精算出真正波段主力的「加權防守價 (Net VWAP)」。當股價大於這個防守價，代表主力帳面上是獲利的（掌控發球權）；若乖離率過高（例如 >10%），要提防主力隨時割韭菜。若股價跌破防守價，代表連主力都被套牢，進入弱勢防守。
* **線性迴歸通道 (LR Channel)**：
    * **極度過熱區（碰觸或突破上軌）**：股價漲速過快，乖離過大，極容易引發獲利了結賣壓。
    * **強勢多頭區（中軌與上軌之間）**：最健康的爬升區間，順勢抱緊。
    * **極度超跌區（碰觸或跌破下軌）**：非理性殺盤區。如果在這裡看到主力瘋狂吃貨，就是送分題級別的買點。
* **核心底單水位**：觀察波段大戶在近 3、10、60 日的淨留倉張數。這是長線保護短線的最強後盾。

## 🕵️ 第二層：中線籌碼 (集保大戶與鎖碼流向)
**【核心任務】**：破解集保結算所的表面數據，還原最真實的「大戶鎖碼含金量」。
* **大戶真實鎖碼率 (C_Value)**：這是 V60 系統的獨家指標。系統會扣除董監事、大股東質押等不會輕易賣出的「死籌碼」，精算出市場上真正在流通的「活籌碼」，再計算波段大戶究竟買走了多少活籌碼。C_Value 越高，代表外面的籌碼越乾淨，拉抬越輕鬆。
* **波段籌碼流向 (純淨大戶變動)**：系統會抓出每週大戶持股的增減，並**強制扣除由當沖或短線客造成的「虛胖張數」**。
    * 當你看到「大戶實質持股增加」，代表主力正在默默鎖碼。
    * 當你看到「散戶增，大戶實質倒貨」，這就是標準的**逃命波**訊號。

## ⚔️ 第三層：短線肉搏 (今日聰明錢與散戶對決)
**【核心任務】**：利用分點足跡，解剖今天的紅黑 K 棒背後，是誰在買、誰在賣。
* **今日主力動向 (聰明錢淨流)**：系統會自動為各大分點貼上標籤（如：波段鐵粉、常駐造市、短線狙擊、快閃散戶等）。這裡只統計「有影響力的大戶（聰明錢）」今天的淨買賣張數。
* **買賣力道對決**：
    * **買方火力倍數**：大單買進平均張數 / 大單賣出平均張數。大於 1.5 倍代表大戶用火力直接壓制空方。
    * **買賣家數差**：買超家數減去賣超家數。**這是極度關鍵的反指標**。家數差為「正」，且數字很大，代表籌碼散落到眾多散戶手中（散戶接刀）；家數差為「負」，代表多數人在賣，但籌碼被少數大戶集中收走（主力吸籌）。

## 🎯 第四層：綜合兵推與最終操作定調
**【核心任務】**：將前面四層的線索交織，給你一個「懶人包」級別的無腦操作結論。系統會嚴格判定多空共振，並產出以下幾種強烈指令：
* 🚨 **【形態轉弱 / 主力撤退，立刻停損】**：技術面破底，加上聰明錢倒貨，不要猶豫，立刻拔檔。
* 🔥 **【形態突破 / 主力點火，強勢追擊】**：底部成型，大戶集結，短線買盤爆發，這是極高勝率的做多訊號。
* 🏃 **【高檔派發 / 趨勢反轉，準備逃命】**：股價雖然還在高檔，但大戶已經在偷偷出貨給散戶，請忽略靜態支撐，先跑再說。
* 💰 **【通道過熱 / 逢高派發，準備停利】**：股價觸及通道上軌，短線主力開始收割，切忌追高，可逢高停利。
* 🛒 **【超跌反轉 / 左側掃貨，絕佳買點】**：股價暴跌觸及下軌，但中線大戶沒跑，短線主力還在趁亂撿便宜，這是勇敢買進的時刻。
* 🛡️ **【強勢推升 / 主力鎖碼，順勢抱緊】**：籌碼高度集中，買方火力強大，乖乖抱緊，不要被小震盪洗下車。
* ⏳ **【高檔震盪 / 壓盤洗盤，觀察防守】**：主力在洗浮額，只要沒跌破「加權防守價」，就持股續抱觀望。

## 📊 三、平日戰情追蹤矩陣：微觀籌碼連續劇

這是你每天必看的連續劇，記錄近 5~60 天每一天的「微觀戰鬥明細」。

**關鍵欄位解析**：
* **聰明錢淨流(張)**：系統過濾掉當沖客、隔日沖後，**真正屬於「波段大戶」與「官股」的淨買賣張數**。這是研判趨勢最核心的指標。
* **大戶買均價**：當天聰明錢買進的平均成本。
* **均價落差**：`當天收盤價 - 大戶買均價`。正數代表大戶當天賺錢（強勢），負數代表大戶當天接滿手血（弱勢）。
* **活躍家數**：當日參戰的分點總數。數字暴增通常意味著當沖熱度過高，籌碼轉趨混亂。
* **買賣家數差**：負數最佳，代表籌碼集中；正數代表散戶螞蟻搬家（籌碼發散）。
* **潛在賣壓(張)**：系統抓出今天進場的「隔日沖」與「短線客」總買超張數。這些籌碼有高達 90% 的機率在明天開盤半小時內被市價砸出。這個數字越大，明天早盤越不能追高。

## 📡 四、終極集保籌碼雷達：宏觀趨勢的照妖鏡

每週更新一次，提供大局觀的籌碼分佈。系統採用了「動態門檻引擎」與「董監事扣除邏輯」，呈現出最真實的股權集中度。

**關鍵欄位解析**：
* **大戶精算門檻**：系統不會死板地只看「1000張大戶」。它會根據這家公司的「股本大小」與「當前股價」自動計算。例如高價股或小股本的公司，可能持有 200 張或 400 張就具備絕對控盤權，系統會自動切換抓取標準。
* **純淨活大戶C_Value(%)**：**這是全系統最狠的數據**。傳統軟體的大戶持股包含了一輩子都不會賣的董監事死籌碼。C_Value 是將「大戶原持股」扣除「董監事持股（死籌碼）」後，去計算大戶掌握了多少「市場上真正流通的活籌碼」。
    * 數值越高，代表在外流通的籌碼幾乎都被主力鎖死，稍微點火就能輕易拉出長紅。
* **原始大戶變動(%)**：官方公佈的表面數據增減。
* **隔日沖虛胖(%)**：系統精算週五（或財報截止當日）短線客買進的佔比。
* **純淨大戶變動(%)**：`原始變動 - 隔日沖虛胖`。這才是本週大戶真正吃貨或出貨的淨值。如果原始變動是 +2%，但虛胖是 1.8%，那純淨變動其實只有 +0.2%，這就是標準的「集保騙局」。
* **總人數變動率(%)**：散戶的溫度計。人數大幅增加（超過 2%）且純淨大戶變動為負，就是最典型的「主力倒貨給散戶」，為強烈逃命訊號。
