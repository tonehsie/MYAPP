🧠 深度診斷總結的「五大核心邏輯」與「最終定調」

這份報告固定會產出五個維度的深度解析，最後給出一個明確的操作建議。

一、 短線戰鬥多空定調（今日籌碼真偽）

這是系統的「照妖鏡」，專門用來抓出市場上最容易騙散戶上車的假象。

抓出「假外資」：系統會比對「三大法人買賣超」與「地方大戶分點」，如果發現高度重疊，會直接警告您這可能是「特定券商偽裝的外資」，隔天極可能倒貨（隔日沖），千萬別看到外資大買就衝進去。

抓出「假主力 / 融資鎖碼」：如果聰明錢大買，但融資也異常暴增，系統會判定這是高槓桿當沖客或短線客在鎖碼。這種籌碼極度不穩定，一旦開盤不如預期就會引發多殺多。

判定「真實掃貨」vs「真實出貨」：

當聰明錢流入 + 買賣家數差為負：代表少數大戶正在瘋狂吃掉多數散戶的賣單，這是標準的「波段起漲」訊號。

當聰明錢撤退 + 買賣家數差為正：代表主力趁高檔倒貨，而散戶正在滿心歡喜地接刀，系統會強力警告您停損。

二、 核心防守價位與安全邊際確認

系統會剔除掉當沖、避險造市等雜訊，幫您精算出真正的「大戶純淨防守價（Net VWAP）」，並計算現在股價距離大戶成本有多遠（乖離率）。

乖離 > 5%（獲利豐厚）：大戶目前賺很多，就算洗盤下殺也不會傷到本錢，您只要跟著均線續抱就好。

乖離 0% ~ 5%（黃金建倉點）：股價剛好貼合大戶成本，這是左側潛伏最愛的甜美買點，因為大戶絕對有極強的動機在這裡護盤。

乖離 < 0%（破線危險）：系統會亮紅燈警告。這代表連砸重金的大戶都被套牢了！一旦大戶決定停損，賣壓會如海嘯般湧出，這時嚴禁凹單。

三、 潛在市場盲點與套利干擾排除

這區專門排除會干擾判斷的「非正常交易」因素。

可轉債（CBAS）套利干擾：有時候您會看到主力大舉賣超，嚇得趕快賣出。但系統如果偵測到 CBAS 餘額同步下降，會告訴您這只是法人在「賣老股、換新債」的無風險套利，並非真的看壞後市。

處置股（關緊閉）警告：系統會計算近 5 日週轉率，若快要踩到交易所的「處置紅線」，會提醒您明天主力極有可能刻意「縮手壓盤」來降溫，讓您不會因為隔天量縮不漲而恐慌。

四、 平日戰情追蹤矩陣 (近 15 日) 趨勢解碼

跳脫單日進出，用「半個月」的視角來看主力在幹嘛。

看「留倉」與「火力」：系統會計算近 10 日核心主力的淨留倉張數，以及今天的「買方火力（買均量除以賣均量）」。

實戰解讀：如果近半個月主力穩定「囤貨」且今天火力大於 1.2 倍，代表盤勢完全由多方掌控；反之，若半個月都在「倒貨」且買盤無力，任何反彈系統都會判定為「逃命波」。

五、 一週集保籌碼雷達（大戶存量與流量雙解碼）

結合每週末公布的集保戶權分散表，系統將其分為「存量（大戶原本持股多寡）」與「流量（本週買賣增減）」。

C_Value（控盤集中度）：系統會扣除死籌碼（董監事不拿出來賣的股票），算出在市面上的流通籌碼中，大戶掌控了多少。

實戰解讀：最好的情況是「高度鎖碼（存量高）＋ 持續吸籌（流量為正）」；最危險的是「籌碼渙散（存量低）＋ 持續拋售（流量為負）」。

🚀 最終操作定調（行動指南）

經過上面五大邏輯的嚴密運算後，系統會結合 「AI 幾何形態（W底/M頭/突破）」 與 「線性迴歸通道（判斷過熱或超跌）」，給出以下 11 種極度明確的實戰指令（擷取常見幾種）：

【形態突破 / 主力點火，強勢追擊】：技術面突破頸線，且聰明錢大舉買入，高勝率買點，建議順勢抱緊。

【軋空噴出飆股模式 / 通道已失效】：股價雖然過熱突破通道上軌，但主力重砲持續狂炸。系統會告訴您此時不要用通道判斷壓力，改用短均線防守，讓獲利奔跑。

【通道過熱 / 逢高派發，準備停利】：股價頂到上軌，且聰明錢開始「趁高檔撤退」。系統會建議您逢高停利，嚴防主力順勢出貨。

【超跌反轉 / 左側掃貨，絕佳買點】：股價跌到下軌，但中線大戶籌碼安定，且今日短線聰明錢大舉淨流入。代表主力趁暴跌吃貨，是極佳的左側買點。

【形態轉弱 / 主力撤退，立刻停損】：技術面跌破頸線或轉弱，且聰明錢果斷撤退。雙重轉空，系統會毫不留情地建議您「立刻停損逃命，嚴禁留戀」。

【兵敗如山 / 全面套牢，嚴禁摸底】：跌破防守價，且短中長線主力全面大舉倒貨。系統會警告此處的任何反彈都是逃命波，嚴禁接刀。
