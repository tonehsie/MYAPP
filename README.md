# 📚 全息量化系統：四大核心模組終極實戰說明書

這份說明書將徹底拆解系統中最核心的四個量化模組。這不是普通的看圖軟體，而是一套「具備邏輯推演能力」的籌碼透視引擎。請將此視為你的實戰操作準則。

---

## 🎯 一、【階段判定】模組：動態風險報酬測量儀

這不是預測未來的摸骨算命，而是根據「最新股價」與「純淨大戶鐵板防守價」的相對位置，精算出來的**當下盤勢結構與風險位階**。

**核心計算邏輯**：
系統會先自動剔除「隔日沖」與「游擊客」的雜訊，算出真正打算做波段的主力「平均建倉成本（鐵板防守價）」。接著，計算目前股價距離這個防守價有多遠（即「乖離率」），來判定目前的位階。

**六大階段詳解與實戰應對**：

* **🟢 主力吃貨中 (安全建倉區)**
    * **觸發條件**：股價在防守價之上，且乖離率在 `0% ~ 10%` 之間。
    * **背後意義**：波段主力正在默默吸籌碼、洗盤。帳面獲利還很小，根本不夠出貨塞牙縫。下檔有主力自己的成本撐著，上檔還沒發動。
    * **實戰操作**：**最佳潛伏期**。適合分批建倉，停損點直接設在「防守價」下方一點點，用極小的風險去搏未來的波段利潤。
* **🔥 趨勢推升 (波段多頭起漲)**
    * **觸發條件**：乖離率落在 `10% ~ 50%` 之間。
    * **背後意義**：主力已經點火拉抬，股價順利脫離成本區，進入主升段。
    * **實戰操作**：**抱緊處理**。這時候不要隨便被洗下車，只要「平日戰情矩陣」中的聰明錢沒有出現連續大賣，就順勢操作，讓利潤奔跑。
* **⚠️ 高檔派發/過熱風險 (乖離破表)**
    * **觸發條件**：乖離率大於 `50%`。
    * **背後意義**：股價已經極端噴出，主力帳面獲利極度豐厚。這個位階主力隨時有誘因啟動「收割倒貨」模式。
    * **實戰操作**：**高度警戒**。嚴禁盲目追高，持股者應開始分批停利。若此時「鷹眼報告」顯示聰明錢淨流出，請立刻清倉。
* **🩹 主力防守戰 (跌破邊緣)**
    * **觸發條件**：股價微幅跌破防守價，乖離率在 `0% ~ -5%` 之間。
    * **背後意義**：主力陷入帳面虧損，防線受到挑戰。這是邊緣試探期，主力可能會在接下來幾天發動逆襲護盤。
    * **實戰操作**：**停看聽**。空手者切勿進場接刀；持股者給予 3 天的觀察期，若主力未防守拉回，需準備執行停損。
* **💀 主力套牢 / 棄守多殺多**
    * **觸發條件**：股價深度跌破防守價，乖離率小於 `-5%`。
    * **背後意義**：波段主力已被嚴重套牢，或者乾脆停損棄守。防線全面崩潰，極易引發散戶與主力的多殺多恐慌踩踏。
    * **實戰操作**：**無條件避開**。任何反彈都是逃命波，嚴格執行停損紀律。
* **⚪ 籌碼中性 (自然換手)**
    * **背後意義**：系統算不出足夠的波段主力籌碼，代表這檔股票目前沒有「大人」在照顧，純粹是散戶與游擊客的自然交易。

## 🦅 二、AI 鷹眼深度診斷報告：金流與結構的文字解碼

這個模組是 AI 引擎每天盤後自動生成的「盤勢總結」，將複雜的數字轉化為直白的實戰建議，共分為四個維度：

**1. 矩陣金流剖析 (看主力的底牌)**
* **均價落差**：今天主力買進的平均成本，跟今天收盤價的差距。
* **主動鎖碼**：主力今天大買，而且買的均價「低於」收盤價。代表主力買在低點，當天就賺錢，控盤底氣極強。
* **接刀套牢**：主力今天大買，但買的均價「高於」收盤價。代表主力進場護盤卻被套，明天如果開盤不強，這些護盤籌碼會變成停損賣壓。
* **拉高派發**：今天股價收紅（漲），但聰明錢卻是「大賣」。這是最經典的出貨手法，利用散戶追高的熱度，主力偷偷把貨倒給散戶。

**2. 火力與籌碼結構剖析 (看戰場的擁擠度)**
* **活躍家數**：今天有交易這檔股票的券商分點數量。越多代表越發散、越熱門（通常是當沖客太多）。
* **買賣家數差**：買方家數減去賣方家數。如果是負數（賣家多、買家少），代表籌碼流向少數大戶手中，是好現象。
* **大戶火力壓制**：計算買方平均每家買的張數，對比賣方平均每家賣的張數。如果倍數超過 1.5 倍，代表有特定大戶用絕對的資金優勢在集中掃貨。

**3. 主力潛伏微觀剖析 (看玩家的屬性)**
* 掃解今天買超前 15 大的分點屬性。如果都是「長駐波段主」、「潛伏造市者」，代表底盤穩固；如果充斥著「隔日沖」、「游擊過客」，系統會發出**【游擊客炸彈】**警告，預告明天早盤必定有倒貨潮。

**4. 週末集保雷達防護 (獨家破謊機制)**
* 每週六公佈的大戶持股比例常常騙人（例如週五隔日沖大買，會讓大戶比例短暫暴增）。此防護網會比對「原始大戶增減」與「隔日沖虛胖佔比」，抓出假買盤，避免你在下週一成為接盤俠。

## 📊 三、平日戰情追蹤矩陣：微觀籌碼連續劇

這是你每天必看的連續劇，記錄近 5~60 天每一天的「微觀戰鬥明細」。

**關鍵欄位解析**：
* **聰明錢淨流(張)**：系統過濾掉當沖客、隔日沖後，**真正屬於「波段大戶」與「官股」的淨買賣張數**。這是研判趨勢最核心的指標。
* **大戶買均價**：當天聰明錢買進的平均成本。
* **均價落差**：`當天收盤價 - 大戶買均價`。正數代表大戶當天賺錢（強勢），負數代表大戶當天接滿手血（弱勢）。
* **活躍家數**：當日參戰的分點總數。數字暴增通常意味著當沖熱度過高，籌碼轉趨混亂。
* **買賣家數差**：負數最佳，代表籌碼集中；正數代表散戶螞蟻搬家（籌碼發散）。
* **潛在賣壓(張)**：系統抓出今天進場的「隔日沖」與「短線客」總買超張數。這些籌碼有高達 90% 的機率在明天開盤半小時內被市價砸出。這個數字越大，明天早盤越不能追高。

## 📡 四、終極集保籌碼雷達：宏觀趨勢的照妖鏡

每週更新一次，提供大局觀的籌碼分佈。系統採用了「動態門檻引擎」與「董監事扣除邏輯」，呈現出最真實的股權集中度。

**關鍵欄位解析**：
* **大戶精算門檻**：系統不會死板地只看「1000張大戶」。它會根據這家公司的「股本大小」與「當前股價」自動計算。例如高價股或小股本的公司，可能持有 200 張或 400 張就具備絕對控盤權，系統會自動切換抓取標準。
* **純淨活大戶C_Value(%)**：**這是全系統最狠的數據**。傳統軟體的大戶持股包含了一輩子都不會賣的董監事死籌碼。C_Value 是將「大戶原持股」扣除「董監事持股（死籌碼）」後，去計算大戶掌握了多少「市場上真正流通的活籌碼」。
    * 數值越高，代表在外流通的籌碼幾乎都被主力鎖死，稍微點火就能輕易拉出長紅。
* **原始大戶變動(%)**：官方公佈的表面數據增減。
* **隔日沖虛胖(%)**：系統精算週五（或財報截止當日）短線客買進的佔比。
* **純淨大戶變動(%)**：`原始變動 - 隔日沖虛胖`。這才是本週大戶真正吃貨或出貨的淨值。如果原始變動是 +2%，但虛胖是 1.8%，那純淨變動其實只有 +0.2%，這就是標準的「集保騙局」。
* **總人數變動率(%)**：散戶的溫度計。人數大幅增加（超過 2%）且純淨大戶變動為負，就是最典型的「主力倒貨給散戶」，為強烈逃命訊號。
